# Справочник сетевого администратора

Мобильное приложение для быстрого доступа к справочной информации по сетевым технологиям.

## 📱 Функции
- 8 категорий (IP-адресация, роутеры, диагностика, безопасность и др.)
- 160+ справочных тем
- Подсветка синтаксиса команд
- Поиск по ключевым словам
- Офлайн-работа (все данные локально)
- Тёмная тема
- Копирование команд в буфер обмена

## 🛠️ Технологии
- Java
- Android SDK (API 24+)
- Markwon (Markdown)
- Gson (JSON)
- Material Design

## 📥 Установка
Скачайте APK из раздела Releases или соберите проект в Android Studio.
